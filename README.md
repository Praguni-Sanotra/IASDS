# Intelligent Academic Scheduling and Decision-Support System (IASDS)

A production-ready platform designed for robust academic scheduling and decision support.

## Project Structure
- `frontend/`: Next.js 14 App Router, TypeScript, Tailwind CSS
- `backend/`: Node.js 20, Express, TypeScript
- `scheduler/`: Python 3.11, FastAPI (AI/CSP Core)
- `nginx/`: Reverse proxy configuration

## Quick Start
1. Clone the project.
2. Copy `.env.example` to `.env` and fill in the values.
3. Start the application stack with:
   ```bash
   docker-compose up --build -d
   ```
